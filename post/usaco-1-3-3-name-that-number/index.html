USACO 1.3.3: Name That Number

张慕晖
2018-08-22

题意

见洛谷 P3864。

分析

简单的映射题。如果从编号出发计算所有可能的字母组合，然后再去字典里查找，那么复杂度会变得过高。题解中给出了一种利用二分查找进行优化的方法，但是不太好懂。显然把字典中的单词映射成号码，再根据号码去查找会简单（且好写）很多。
